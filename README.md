# ABCLearn.app

ABCLearn.app 是一个用于 【AI 驱动的在线学习平台】的应用项目，旨在为用户提供高效、直观的学习体验。

---

## ✨ 功能特性
- **智能学习推荐**：根据用户的学习习惯和进度，自动生成学习路径
- **交互式课程**：支持视频、音频、测验等多种学习形式
- **多终端支持**：可在 Web、桌面端和移动端无缝切换
- **实时数据统计**：提供学习进度与效果分析
